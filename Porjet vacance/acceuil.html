Equipe

Voici les différents équipes qui participent a ce tournoix de futsal/p>

  • Les match se jouent le mercredi a partir de 12h45

Classement

Le classement est mise a jour chaque semaine

  • La différence de but est importante pour le classement finale
  • Une victoire vosu donne 3 points
  • Un match nul vous donne 1 point
  • Une défaite vous donne 0 point

    • Voici le déroulemeznt des match et du classement .

  • Chaque mercredi un match a lieux au gymnase a partir de 12h45
  • Les équipe s'affronte et Loic note le score pour ensuite lettre a jour le classement des équipes
